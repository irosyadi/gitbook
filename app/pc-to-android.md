# PC to Android Connection

## PC - Android Connection

- KDE Connect by KDE:
    - Android to PC:
        - Send files
        - Send clipboard
        - Slideshow remote
        - Multimedia control
        - Remote input (as wireless mouse)
        - Run (Windows) command
        - Notification Sync
        - Telephony Notifier (call)
    - PC to Android:
        - Send SMS
        - Media Player Control
        - Remote keypresses from keyboard (as wireless keyboard to Android)
        - Mouse receiver (as mouse for Android)
        - Run (Windows) command
        - Find device
        - Send files
    - Cons:
        - should be in same local networks
        - bug in find phone
- Join by JoaoApps:

        - Send note
